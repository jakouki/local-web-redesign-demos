# local-web-redesign-demos

Small static demo sites for local company website redesign proposals.

## Demo URL structure

After GitHub Pages is enabled, each company can receive a separate URL:

- Kensosha: `https://jakouki.github.io/local-web-redesign-demos/kensosha-demo/`
- Yuwa House: `https://jakouki.github.io/local-web-redesign-demos/yuwa-demo/`
- Takizawa Electric Works: `https://jakouki.github.io/local-web-redesign-demos/takizawa-demo/`
- Kyokuo: `https://jakouki.github.io/local-web-redesign-demos/kyokuo-demo/`
- Hanz: `https://jakouki.github.io/local-web-redesign-demos/hanz-demo/`
- Iijima: `https://jakouki.github.io/local-web-redesign-demos/iijima-demo/`
